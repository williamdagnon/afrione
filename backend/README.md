# Backend Node.js (Express) – Futuristia

Ce backend fournit une API REST pour la gestion des utilisateurs, produits, achats et notifications.

- Authentification JWT
- Connexion PostgreSQL (compatible Supabase)
- Documentation Swagger

## Démarrage rapide

```bash
npm install
npm run dev
```

## Structure prévue
- `/src` : code source (routes, contrôleurs, modèles)
- `/config` : configuration (DB, JWT, etc.)
- `/docs` : documentation Swagger

## À faire
- Ajouter vos variables d'environnement dans `.env`
- Adapter la connexion à Supabase si besoin
